Request for Discussions is a series of documents related to Free
Software and free society.

All documents first go through the DRAFT status.  When they become RFDs,
they are permenantly immutable, except for the Updated By and Obsoleted
By lines.  Errors corrections must be submitted through new RFDs and
obsolete the erroneous one for large corrections, or by an ERRATA file
in the RFD's directory.

An informational RFDs describing the document series is upcoming.

Currently, IETF RFC tools are used for typesetting.  An experimental
tool is being developed in utilities/.
